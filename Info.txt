Uma interface onde o usuário posssa selecionar as informações de Monster Hunter Freedom Unite.
As seguintes informações devem ser acessadas pelo usuário:
    - Lista de Quests e as Key Quests.
        - Elder Quests
        - Nekoht Quests
        - Guild Hall Quests

    - Lista de Monstro:
        - Nome do Monstro em Inglês & Japones.
        - Classe do Monstro
        - Elemento & Fraqueza Elemental
        - Habitat
        - Geração
        - Frase*

    - Lista de Items:
        - Nome, Informação e Preço


Saida
    Um App com as informações escolhidas pelo usuário.

Processo
    Lista de Quests e as Key Quests.
        Um botão onde o usuário seleciona as quests pelo nome, a lista vai ser com as key quests.
            Base de dados com todas as Key Quests e sua categoria.

    Lista de Monstros
        Um botão onde o usuário vai escolher a lista de monstros digitando seu nome.
            Base de dados com todas os monstros e sua classe.

    Lista de Itens
        Uma listagem com todos os items e suas funções.
            Base de dados com todas os itens seu preço e suas funções.

Entrada
    Monster Hunter Wiki: https://monsterhunter.fandom.com/wiki/Monster_Hunter_Freedom_Unite
    Ferramentas:
        - Python 3.8.0.
        - Jupyter Notebook.
        - Bibliotecas Graficas como PySimpleGUI.
        - Bibliotecas de Webscraping como Beautiful Soup.
